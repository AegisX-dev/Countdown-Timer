# Countdown Timer  

## Overview  
The **Countdown Timer** is a web-based application that allows users to create multiple countdown timers with custom names, dates, and times. The timers run in real-time and notify the user when the countdown reaches zero.  

## Features  
- **Multiple Timers**: Users can add multiple timers on the same page.  
- **Custom Timer Name**: Each timer can be given a name for easy identification.  
- **Real-Time Countdown**: Displays the remaining days, hours, minutes, and seconds.  
- **Start, Stop, and Reset Functionality**: Users can start, pause, or reset a timer anytime.  
- **Delete Timer**: Users can remove a timer when it's no longer needed.  
- **Audio Notification**: A beep sound is played when the countdown reaches zero.  
- **Responsive Design**: Fully responsive layout with a modern and stylish UI.  

## Technologies Used  
- **HTML5**: Structure of the application.  
- **CSS3**: Styling with animations and a visually appealing interface.  
- **JavaScript**: Timer logic, event handling, and DOM manipulation.  

## How to Use  
1. **Enter Timer Name** *(optional)*: Type a name in the text input field.  
2. **Set Date and Time**: Use the date and time pickers to set the target countdown time.  
3. **Start Timer**: Click the "Start" button to begin the countdown.  
4. **Pause Timer**: Click the "Stop" button to pause the countdown.  
5. **Reset Timer**: Click the "Reset" button to clear all inputs and reset the countdown.  
6. **Delete Timer**: Click the 🗑️ button to remove a timer.  
7. **Add More Timers**: Click the **"+"** button to create additional timers.  

## Installation & Setup  
1. Clone this repository:  
